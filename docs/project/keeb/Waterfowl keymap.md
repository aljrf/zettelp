# Waterfowl keymap

## Shorter home row mods:

```
#define HRML(k1,k2,k3,k4) &mt LSHFT k1  &ht LALT k2  &ht LCTRL k3  &ht LGUI k4
#define HRMR(k1,k2,k3,k4) &ht RGUI k1  &ht RCTRL k2  &ht RALT k3  &mt RSHFT k4
```

## Home row Mods

|       |      | Altgr |     |     |     | Altgr |      |       |
